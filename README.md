# Accessabled 🔗

**Accessabled** is a suite of web accessibility tools aimed at enhancing user experience for individuals with diverse needs. This project features two powerful browser extensions: **Ecopy** and **TravelBuddy**.

## 🌐 Extensions

### 🚀 1. Ecopy — Speech to Text Form Filler

Ecopy is designed to help users fill out web forms using their voice. It leverages **Speech-to-Text** technology to convert spoken input into written text, enhancing usability for users with motor disabilities or typing challenges.

**Key Features:**
- 🎙️ Speech recognition to fill form fields
- 🧠 Intuitive interface for easy navigation
- 🔁 Supports multiple form types and input fields

---

### 🧭 2. TravelBuddy — Accessibility Assistant for Web

TravelBuddy is an accessibility-focused browser extension offering multiple features to assist users with reading, visual impairments, and cognitive disabilities.

**Key Features:**

- 🔊 **Text-to-Speech**: Converts on-screen text into spoken words.
- 🔠 **Text Sizer**: Adjusts text size for better readability.
- ✂️ **Text Summarization**: Provides concise summaries of long articles and content.
- 🌈 **Color Blind Mode**: Color-blind-friendly themes with:
  - 🔴 Red Mode
  - 🟢 Green Mode
  - 🔵 Blue Mode
